# フォルテス

アイドル × リズム × ターン制RPG。現実の歩行距離と連動し、都市に出現する謎の怪物 **Noise** を討伐して街を解放していく Unity モバイルゲームです。

---

## 作品概要

人々はアイドルとして活動しながら、都市に出現する謎の怪物「Noise」と戦う使命を背負う。音楽ゲームのリズム要素とターン制RPGを融合させたバトルシステムを採用し、楽曲に合わせた操作によってパフォーマンスを高めながら戦闘を進めていく。プレイヤーは各地で発生する Noise を討伐し、人々の希望を取り戻しながら、蝕まれた都市全体の解放を目指す。

---

## 遊び方

### 全体の流れ

```
タイトル → ホーム →（スキャン）→ ステージ選択 → 装備 → バトル → リザルト → ホーム
                ↘ クラフト（装備合成）
```

### 歩行距離とNoise解放

- 端末の **GPS** で歩いた距離を計測し、累計距離から **NoisesAmount** を算出します（例：5,000 m ごとに +1、最大 12）。
- 日次スケジュールが有効な場合、初日は一定の距離が付与され、翌日以降は前日の歩行分が反映されます。
- 距離が増えるほど多くの Noise がマップ・ステージ一覧に出現し、討伐対象が増えていきます。

---

## バトルシステム

### 起動方法

1. Unity Hub で `HackathonGame` フォルダを開く
2. `Assets/Scenes/TitleScene.unity` を開く
3. Play ボタンで実行

モバイル実機では GPS による距離計測が有効になります。エディタ上ではデバッグ用の距離設定（`DistanceDebugScene` 等）を利用できます。
